# Sport Calendar

The **Sports Calendar Application** is a web-based system for managing and viewing sports events such as matches between teams.  

## Features

- View event list with pagination and sorting by sport type and date of event
- Create new events using a form
- View detailed information about particular event
- Validate input data on both frontend and backend
---

## Technologies Used

- **Frontend:** HTML5, CSS3, JavaScript  
- **Backend:** Java 21, Spring Boot
- **Database:** PostgreSQL  
- **Build Tool:** Maven 
- **Testing:** JUnit, Mockito, Testcontainers
- **Containerization:** Docker
- **API Documentation:** Swagger (OpenAPI)
